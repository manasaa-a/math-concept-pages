What is a Horizontal Asymptote?

Think about it

Every hour, your body clears 30% of whatever painkiller remains. The amount keeps halving — but can it ever hit exactly zero? Is there a line on the graph it's forever chasing?

A horizontal asymptote is a horizontal line y = k that a function approaches as x → ∞ or x → −∞, without necessarily reaching it. It tells you where the curve is headed in the long run — not where it is at any specific point.

Formally: if limx→∞ f(x) = k or limx→−∞ f(x) = k, then y = k is a horizontal asymptote of f.

A function can have at most 2 horizontal asymptotes — one as x → +∞ and a different one as x → −∞. And crucially: unlike vertical asymptotes, a curve can cross its own horizontal asymptote.

The Three Rules for Rational Functions

Let n = degree of the numerator, d = degree of the denominator. Every rational function falls into exactly one case.

Rule
deg(numerator) < deg(denominator) → Horizontal asymptote: y = 0

Why it works: As x grows huge, the larger denominator overwhelms the top, driving the fraction toward zero.

Example 1
f(x) = 3 / (x + 2)
deg top = 0, deg bottom = 1
HA: y = 0
Example 2
f(x) = 5x / (x² + 1)
deg top = 1, deg bottom = 2
HA: y = 0
Rule
deg(numerator) = deg(denominator) → HA: y = a/b where a, b are leading coefficients

Why it works: At extreme x, all lower-power terms vanish — only leading terms survive, so the limit equals their ratio.

Example 1
f(x) = 2x / (x − 3)
Leading coeffs: 2 and 1
HA: y = 2
Example 2
f(x) = (6x² + 1) / (3x² − 5)
Leading coeffs: 6 and 3
HA: y = 2
Rule
deg(numerator) > deg(denominator) → No horizontal asymptote

Why it works: The numerator grows faster than the denominator, so the function heads toward ±∞ and never settles.

Example 1
f(x) = (x² + 1) / (x + 3)
deg top = 2 > deg bottom = 1
No HA
Example 2
f(x) = x³ / (x² − 4)
deg top = 3 > deg bottom = 2
No HA
Rule
f(x) = ab^(kx) + c → HA: y = c (the vertical shift)

Why it works: As x → ∞ or −∞, the exponential term ab^(kx) collapses to 0, leaving only c. This is exactly the medicine model — concentration approaches y = 0 (where c = 0) forever.

Example 1
f(x) = 3 · 2^x
c = 0 (no shift)
HA: y = 0
Example 2
f(x) = 4^x + 2
c = 2
HA: y = 2
Mini-check

The function f(x) = (5x² − 3) / (2x² + 1) has degree 2 on top and degree 2 on the bottom. What is its horizontal asymptote? Divide the leading coefficients to check.

Key Takeaways

TermDefinition
Horizontal Asymptote A horizontal line y = k that f(x) approaches as x → ±∞. Defined by limx→±∞ f(x) = k.
Degree The highest power of x in a polynomial. Comparing numerator and denominator degrees determines which HA rule applies.
Leading Coefficient The coefficient of the highest-degree term. When n = d, the HA equals the ratio of leading coefficients (a/b).
Rational Function A function of the form p(x)/q(x) where p and q are polynomials. All three degree-comparison rules apply to these.
Exponential Decay A function of the form f(x) = ab^(kx) + c. Its HA is always y = c — the value the exponential term approaches as it vanishes.

Frequently Asked Questions

A horizontal asymptote is a horizontal line y = k that a curve approaches as x heads toward positive or negative infinity. It describes long-run behavior — not where the function is at a specific point, but where it is forever trending.
A vertical asymptote (x = k) marks where a function blows up toward ±∞ — the function is undefined there. A horizontal asymptote (y = k) marks where the function settles at extreme x values. Key difference: a curve can cross a horizontal asymptote; it can never cross a vertical one.
Compare degrees of numerator (n) and denominator (d). If n < d: HA is y = 0. If n = d: HA is the ratio of leading coefficients. If n > d: no HA exists. These three cases cover every rational function.
Yes — and this surprises most students. A horizontal asymptote only governs behavior at infinity. Before that, the curve can cross y = k any number of times. For example, f(x) = sin(x)/x has HA y = 0 but crosses the x-axis infinitely many times.
For f(x) = ab^(kx) + c, the HA is always y = c. The vertical shift c is the value the exponential approaches but never reaches. The base form f(x) = b^x has HA y = 0.
At most two — one as x → +∞ and a different one as x → −∞. This happens with certain square-root functions like f(x) = (x+1)/√(x²−1), which has HA y = 1 going right and y = −1 going left. Rational functions with real coefficients have at most one.
No. Linear, quadratic, cubic — all polynomials grow without bound as x → ±∞ and never settle toward a finite line. Similarly, basic trigonometric functions and logarithms have no horizontal asymptotes.
At very large x, lower-degree terms become negligible — only the highest-power terms matter. Dividing those gives the limit. If the numerator grows faster (n > d), the limit is ∞. If the denominator dominates (n < d), the limit is 0. If they match, the ratio of leading coefficients is what survives.
