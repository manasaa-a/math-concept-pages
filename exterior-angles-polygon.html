Regular polygons — a useful table

For a regular polygon with n sides: each exterior angle = 360° ÷ n, each interior angle = 180° − exterior.

PolygonSidesEach exteriorEach interior
Equilateral triangle3120°60°
Square490°90°
Regular pentagon572°108°
Regular hexagon660°120°
Regular octagon845°135°
9 sides. n = 360° ÷ 40° = 9. It's a regular nonagon. You can always find the number of sides by dividing 360° by each exterior angle.

Quick reference

Key takeaways

Interior angleThe angle inside the polygon at each vertex.
Exterior angleThe supplement of the interior angle. Interior + exterior = 180°. It's the turn you make when walking the boundary.
Sum of exterior anglesAlways 360° for any convex polygon. Walking the full boundary returns you to your starting direction — one full rotation.
Each exterior angleFor a regular polygon with n sides: 360° ÷ n.
Finding n from anglesIf each exterior angle is known: n = 360° ÷ each exterior angle.
